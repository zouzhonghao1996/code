CSS（Cascading Style Sheets）中文名为“层叠样式表”，用于设置网页中各标签的样式。
“样式”是指网页中文本的大小、颜色以及图片的边框、位置等。
“层叠”是指当在HTML文件中引用多个样式文件时，浏览器将依据层叠顺序处理，以避免发生冲突。


引入CSS的方法有以下几种:
1.将CSS样式表放置在HTML文件头部：内部样式表(嵌入式样式)。
2.将CSS样式表放置在HTML文件主体：内联样式(行内样式表)。
3.将CSS样式表放置在HTML文件外部：链接样式表(外联样式表)

CSS语法格式:
选择器名 { 
  属性名: 属性值;
  属性名1: 属性值1;
  属性名2: 属性值2;
  ...
}

常用的选择器:
标签选择器 / 类选择器 / id选择器
